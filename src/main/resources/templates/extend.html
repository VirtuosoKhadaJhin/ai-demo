Smart photo library

基于Amazon Rekognition的图片文字识别案例.

Amazon Rekognition 提供了用于检测图像中的标签和人脸的非存储 API 操作。 标签是基于其内容在图像中找到的对象、场景或概念。 例如，热带海滩上的人物照片可能包含人物、水、沙、棕榈树和泳装 (对象)、海滩 (场景) 以及户外 (概念) 等标签。
